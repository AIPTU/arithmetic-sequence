# Urutan Aritmatika

Sebuah aplikasi web untuk membuat dan menyelesaikan urutan aritmatika.

## Fitur

- Membuat urutan aritmatika
- Menyelesaikan urutan aritmatika yang hilang

## Teknologi

- React
- Framer Motion
- Vite
